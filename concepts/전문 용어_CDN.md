# CDN(Content Delivery Network, 콘텐츠 전송 네트워크)

_이미지나 파일을 전 세계에 빠르게 뿌려주는 배달망이다._

예를 들어, localhost:3000/uploads/image.jpg 이런 식으로 이미지가 개발 서버에서 직접 내려가지만, 실제 서비스가 커지면 문제가 생긴다. 사용자가 한국뿐만 아니라 해외에서도 접속할 수 있고, 이미지 파일이 많아지면 서버가 느려지거나 다운될 수 있다. 그래서 CDN을 쓰면 이렇게 된다.

- 이미지를 **AWS S3**, **Cloudflare**, **Akamai**, **Naver Cloud** 같은 CDN 서버에 올려둔다.
- 전 세계 여러 지역에 **캐시 서버(복사본 서버)**가 생긴다.
- 사용자는 자기 위치에서 **가장 가까운 서버**로부터 이미지를 받아오게 된다.

**즉, 빠르고 안정적으로 이미지 로드가 가능해진다.**
**이미지·동영상 같은 정적 파일을 전 세계 어디서든 빠르게 제공하기 위한 분산 서버 네트워크**

_활용 예시_

- .env에 BASE_URL=https://cdn.example.com
- 서버에서는 BASE_URL을 기준으로 이미지 URL을 만들어 준다.

**나중에 CDN을 바꾸더라도 코드 수정 없이 .env만 바꾸면 끝이다.**
